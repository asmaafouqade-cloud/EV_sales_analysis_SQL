                                              EV Market Analysis with SQL

This project analyzes global electrical vehicle sales using SQL

Note : The insights are specific to this dataset and do not necessarily reflect real-world EV market trends

Objectives:
- Practice SQL for data cleaning and exploratory analysis
- Analyze trends in electric vehicle sales
- Understand factors influencing EV adoption
- Compare market performance across regions and brands
- Explore the environmental impact of different vehicle types
- Derive insights from data using SQL queries

Tools : MySQL workbench

Dataset :
EV sales dataset retrieved from Kaggle including :Region, Country, Brand, EV sales, EV prices, Government incentives, Co2 reductions

Business Questions :
- Which countries sell the most EVs?
- Which brands dominate the market?
- Do government incentives increase adoption?
- Do cheaper EVs sell more?
- Which regions have the highest adoption?
- Which vehicles reduce the most CO₂?

Key insights
- Canada records the highest EV sales in this dataset.
- Hyundai and Kia appear as leading brands.
- Lower-priced EV brands tend to sell more units.
- Europe shows the highest adoption rates.
- Government incentives are associated with higher adoption.
- SUVs show the highest CO₂ reduction on average.
